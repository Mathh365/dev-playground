# Exercícios de Programação em C#

Exercícios passados em sala de aula para serem feitos em Java, inclusive podem ser encontrados na pasta **ExerciciosEmJava** logo abaixo.  
Porém, para praticar minhas habilidades de C#, que é a linguagem que pretendo dominar, resolvi fazê-los também pelo **.NET**.

O exercício exibido abaixo é o quinto dessa sequência, que apresenta um menu interativo no console.  
Esse menu permite que o usuário escolha diferentes opções, como exibir um "Hello World", imprimir uma sequência numérica personalizada ou encerrar a aplicação. A navegação funciona de forma simples: o usuário digita o número correspondente à função desejada e o programa executa a ação.

##  Sobre o Ex05

O exercício 05 implementa:

- Um **menu interativo** em loop
- Leitura de opções numéricas
- Execução de funções específicas para cada escolha
- Validação simples com `switch`
- Funções auxiliares (`PrintNumeros`, `Loading`, `Pausar`)

É uma prática ótima para reforçar:
- Estrutura de controle
- Separação lógica por métodos
- Fluxo de execução e retorno ao menu principal
